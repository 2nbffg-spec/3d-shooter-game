# 3D Shooter Game

A fun, free 3D top-down shooter game built with Three.js and hosted on GitHub Pages. Play directly in your browser!

## 🎮 Game Features

- **3D Top-Down View** - Dynamic camera following the player
- **Enemy AI** - Enemies move toward you and shoot back
- **Multiple Levels** - Progressive difficulty with more enemies per level
- **Health & Score System** - Track your progress and score
- **Particle Effects** - Visual feedback for explosions and impacts
- **Smooth Gameplay** - Optimized performance for smooth 60fps

## 🕹️ Controls

| Key | Action |
|-----|--------|
| **WASD** | Move forward, back, left, right |
| **Mouse Move** | Look around |
| **Click** | Shoot |
| **Space** | Jump |

## 🎯 Gameplay

1. Shoot enemies before they reach you
2. Enemies have health and will shoot back
3. Complete waves of enemies to advance to the next level
4. Each level has more enemies and increases in difficulty
5. Game ends when your health reaches 0

## 🚀 How to Play

1. Visit the live game link: [Play Here](https://2nbffg-spec.github.io/3d-shooter-game/)
2. Use WASD to move and mouse to look around
3. Click to shoot enemies
4. Survive as many levels as you can!

## 🛠️ Technologies Used

- **Three.js** - 3D graphics library
- **HTML5** - Markup and structure
- **JavaScript** - Game logic
- **GitHub Pages** - Free hosting

## 📊 Game Stats

- Health: Decreases when enemies hit you or shoot you
- Score: +100 points per enemy defeated
- Level: Increases when you defeat all enemies in a wave
- Enemies: Shows how many enemies are in the current wave
